안녕하세요!
이번 대규모 업데이트인 4.0.0은 레거시 지원을 종료하고
메인 화면에 대한 디자인 업데이트, 뉴스, 설정 리뉴얼 등 많은
디자인 수정이 포함되어있습니다.
 
디자인 레이아웃 수정
- 메인 화면 디자인이 수정되고 설정, 뉴스, 메인 화면이 스왑되는 방식으로 변경됩니다.
- 설정이 여러 탭으로 나누어지게 되었고 테마 기능을 지원할 예정입니다.
- 설정 탭에서 '정보' 탭이 추가되었습니다. 라이센스 정보와
   도움을 주신 분들을 확인 할 수 있습니다.
 
내부 레거시 코드 수정
- 마인유틸 1.0.0의 코드를 수정하였습니다.
- 로그인 코드를 새롭게 리뉴얼하여 더 빠르고 안전해졌습니다.
- 마인크래프트 다운로드 속도가 향상되었습니다.
- DK STUDIOS 서버로 연결되는 http 코드를 대거 수정했습니다.
 
새 UI 적용
- 새 Window 11식 UI가 적용되었습니다.
- 토글 스위치, 슬라이드 바, 아이콘 등이 적용되었습니다
